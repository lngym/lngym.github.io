接下来就可以在下面写 CSS 样式代码了，记得用 <style> 包裹。

  • 基础语法元素

    • 对于关键字和 CSS 选择器部分：变为金黄色，并加粗。

    • 对于内置的函数、类以及字面量：变为棕色。

    • 对于字符串类型、正则表达式以及新增内容：变为浅绿色，加一点背景阴影。

    • 对于数字、元数据与连接：变为棕色。

    • 对于注释及引用：变为深灰色，并且斜体。

  • 变量系统

    • 对于普通变量与模板变量：变为粉红色，并且斜体。

    • 对于语言变量与常量：变为青黄色，并且添加下划线。

  • 函数系统

    • 对于函数名、函数参数列表：变为天蓝色。

    • 对于函数参数变量：变为金色。

  • 类型系统

    • 对于类、结构体与类型：变为金色。

    • 对于模板与泛型：变为蓝青色。

  • 操作符与标点

    • 对于操作符：变为浅紫色，并加粗。

    • 对于标点：变为灰色，并设置 0.75 的透明度。

  • 属性系统

    • 对于 HTML 属性和对象属性：变为浅棕色。

    • 对于符号与列表符号：变为蓝青色。

  • 特殊语法

    • 对于字符串内部的元内容和 HTML/XML 标签名：变为亮红色。

    • 对于命名空间/模块路径：变为紫色，并设置 0.8 的透明度。

  • 装饰性元素

    • 行号设置：变为中灰色，留出右内间距，右对齐并设置禁止选中防止用户误复制。

    • 重点标记：设置浅黄色半透明背景，留出左右内边距，并设置轻微圆角。

    • 删除内容：设置极浅的红色背景。

  • 交互效果

    • 代码块悬停：鼠标悬停时会显示绿色发光边框，并平滑过渡。

    • 文本选中高亮：鼠标选中文本时，会显示带 30% 透明度的浅蓝色背景。
